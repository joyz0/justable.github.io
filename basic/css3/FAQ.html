# formatting context

浏览器渲染时，会生成一个个盒子，每个盒子遵循的规则称为格式化上下文（formatting context），传统的块状盒子都是 block formatting context，grid 和 flex 有属于自己的 formatting context

# 在浮动布局中，height: 100%无效

height: 100%是相对父节点的，浮动中的节点是无法获得父节点高度的
